# homebrew-geogrep

Homebrew tap for [geogrep](https://github.com/jesperfjellin/geogrep) — fuzzy grep for geospatial vector data.

## Install

```bash
brew tap jesperfjellin/geogrep
brew install geogrep
```

## Update

```bash
brew update
brew upgrade geogrep
```

## Source

This tap is mirrored from `homebrew/` in the main [geogrep](https://github.com/jesperfjellin/geogrep) repo. Changes flow main repo → tap repo on each release.
